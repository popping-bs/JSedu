if~else, if for ~ in ~문은 집합 객체에서 요소의 key를 순차적으로 반환합니다
배열에서 key는 index 입니다
for ~of~문은 집합 객체에서 요소의 value를 순차적으로 반환 합니다 (es6)
Object.keys(), Object.values(), Object.entries() 들은 (es8)
destructuring: 배열 또는 객체의 값을 분해해서 변수에 개별 값을 할당 (es6)
spread(...) 연산자: 배열 또는 객체의 요소를 분해 (es6) continue, break 문 지원 됨

q> 객체의 요소를 분해할당 할때 객체에 존재하지 않는 key를 분해 할당 변수로 선언하면?
